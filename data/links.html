CelesTrak

The CelesTrak site is the place to be to find more information about the SGP4 model. It has source code in different programming languages, for people who wants to implement it in there own project. And it also provides up-to-date TLE data sets for almost any satellite.

Source code

The code for this project can be found on Github. It use the Arduino core for the ESP8266.

Similar projects

ISS Orbit Tracking Pointer by Grady Hillhouse. It calculates the ISS position with the SGP4 model and points to the correct direction with a rotating arm. This became the main source of inspiration for my own version.

ISS Overhead by cat-haines. This project indicates when the ISS is overflying. It doesn't calculated the position, but instead it use an API to find the rise and set time.
